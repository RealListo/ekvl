### Elder Kings: VL for CK2 ###
Checksum: CVWG


Thank you for downloading Elder Kings: VL for Crusader Kings II.

This release is compatible with Crusader Kings II version 3.3.X.

## Installation Instructions
1. Download the mod archive.
2. Extract the .zip file to `Documents\Paradox Interactive\Crusader Kings II\mod`.
3. In the extracted folder:
   - Make a copy of `descriptor.mod`.
   - Edit the file if necessary to ensure the `name` and `path` fields match the folder name.
   - Rename and move the `.mod` file to `Crusader Kings II\mod`.
4. Enable the mod in the Launcher.
